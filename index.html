Privacy Policy for SiteBlocker

Last Updated: June 2026

SiteBlocker is a browser extension that helps users block websites and manage browsing habits.

SiteBlocker does not collect, store, transmit, sell, or share personal information with any third party.

The extension stores user preferences, blocked websites, and optional blocking schedules locally on the user's device using Chrome's storage API. This information never leaves the user's device.

SiteBlocker does not use analytics, tracking, advertising, or external servers to process user data.
